# Mediglow — Prototype

This prototype demonstrates a privacy-first, client-side implementation of Mediglow: a cosmetic and medicine safety analysis tool.

Quick start:

1. Serve the folder with a static server (recommended):

```powershell
npx serve .
```

2. Open the site in a mobile browser or desktop and paste an ingredient list, select product type, and click "Analyze".

Notes:
- OCR and barcode scanning are not fully integrated in this prototype (stubs available). The analysis runs on manual ingredient input.
- This is educational only and not medical advice.
